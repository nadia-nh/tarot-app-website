# Arcana Flux Website

A simple landing site for the Arcana Flux tarot reading app. This repository contains the app marketing page, privacy policy page, style sheet, and image assets used to showcase the Android tarot application.

## Features

- App overview page with screenshots and highlights
- Built-in privacy policy section with downloadable page
- Mobile-friendly responsive design
- Static site ready for deployment on GitHub Pages or any static host

## Contents

- `index.html` - Main site landing page with app overview and policy navigation
- `privacy-policy.html` - Dedicated privacy policy page
- `styles.css` - Global styling for the site
- `images/` - Screenshot and illustration assets used on the landing page
- `CNAME` - Custom domain record for GitHub Pages hosting

## Notes

- This site is a static HTML/CSS project and does not require a server.
- The privacy policy content is included both in `index.html` and `privacy-policy.html`.
